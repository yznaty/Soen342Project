## How to run

From the `taskManager` folder:

1. Compile:
   javac -cp ".:lib/sqlite-jdbc-3.51.3.0.jar" src/com/taskmanager/*.java

2. Run:
   java -cp ".:lib/sqlite-jdbc-3.51.3.0.jar:src" com.taskmanager.Main

## Notes
- The SQLite database file (taskmanager.db) will be created automatically on first run.
