## How to run
From the `taskmanager` folder, run:

```bash
find . -name "*.class" -delete
javac -d . *.java
java com.taskmanager.Main
```

## Notes
- Make sure all files are saved before compiling.
- Run these commands inside the folder that contains the `.java` files.
- If compilation succeeds, the CLI menu will appear.
